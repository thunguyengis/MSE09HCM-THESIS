# HT-Chatbot

## 1. Prerequisites
### Backend
```
cd real-estate-LT-chatbot
pip install -r requirements.txt
```
Fill dbstring in ./real-estate-LT-chatbot/.env
### Frontend
```
cd LTBotKit
npm install
```

## 2. Run it
### Backend
```
cd real-estate-LT-chatbot
python api_conversation_manager.py
```
### Front end
```
cd LTBotKit
npm start
```
Go to [http://localhost:3000/](url) and try it!
